```
│  │   │                                                                    │ │

│  │   │                                                                    │ │
│  │   ├── [跳过条件2] 黑名单                                                │ │

│  │   │                                                                    │ │
│  │   ├── 更新统计信息                                                      │ │
│  │   │   ├── messages_added, messages_skipped                             │ │
│  │   │   ├── images_saved, videos_saved                                   │ │
│  │   │   └── voice_messages                                               │ │
│  │   │                                                                    │ │
│  │   ├── 保存检查点                                                        │ │
│  │   │   └── _save_checkpoint(synced_list, stats, total, all_customers)   │ │
│  │   │                                                                    │ │
│  │   ├── [错误处理]                                                        │ │
│  │   │   ├── 记录错误到 stats["errors"]                                    │ │
│  │   │   ├── 设备丢失 → 立即停止                                           │ │
│  │   │   └── 其他错误 → 尝试恢复到私聊列表                                  │ │
│  │   │                                                                    │ │
│  │   └── 人类延迟 (_human_delay("user_switch"))                           │ │
│  └────────────────────────────────────────────────────────────────────────┘ │
│                               │                                             │
│                               ▼                                             │
│  ┌────────────────────────────────────────────────────────────────────────┐ │
│  │ Step 10: 完成清理                                                       │ │
│  ├────────────────────────────────────────────────────────────────────────┤ │
│  │ if no errors:                                                          │ │
│  │   └── _clear_checkpoint() → 删除 sync_checkpoint.json                  │ │
│  │ else:                                                                  │ │
│  │   └── 保留检查点，供重试使用                                            │ │
│  │                                                                        │ │
│  │ 记录最终统计信息                                                        │ │
│  │ return stats                                                           │ │
│  └────────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
│                               ▼                                             │
│  ┌────────────────────────────────────────────────────────────────────────┐ │

│                               ▼                                             │

│  │                                                                        │ │

│  │   │   ├── 内联下载语音 (长按播放→等待缓存→adb pull)                     │ │

│  │   └── 检测结束条件 (屏幕不再变化)                                        │ │
│  │                                                                        │ │
│  │ 返回: ExtractionResult                                                  │ │
│  │   ├── messages: List[ConversationMessage]                              │ │
│  │   └── total_count: int                                                 │ │
│  └────────────────────────────────────────────────────────────────────────┘ │
│                               │                                             │
│                               ▼                                             │
│  ┌────────────────────────────────────────────────────────────────────────┐ │
│  │ 7. 处理和存储消息                                                       │ │

│  │   │                                                                    │ │
│  │   └── processed = await _process_and_store_message(msg, customer)      │ │
│  │       │                                                                │ │
```

### 1.8 当前流程问题总结

| 问题                 | 描述                                                     | 影响           |
| -------------------- | -------------------------------------------------------- | -------------- |
| **流程耦合**         | 入口脚本包含大量业务逻辑（AI服务、边车、回调设置）       | 难以测试、修改 |
| **Monkey-patch**     | 通过运行时替换方法注入功能                               | 脆弱、难以追踪 |
| **回调地狱**         | 语音处理、客户语音、AI回复都通过回调注入                 | 流程不清晰     |
| **状态管理**         | 客户上下文通过闭包变量传递                               | 容易出错       |
| **错误恢复**         | 错误恢复逻辑散落在各处                                   | 不一致         |
| **测试困难**         | 无法单独测试各个组件                                     | 质量难保证     |
| **无实时响应**       | 发送消息后立即退出，无法处理用户即时回复                 | 用户体验差     |
| **未读优先不够灵活** | 只在同步开始时检测未读，同步过程中新增的未读无法优先处理 | 响应不及时     |

### 1.9.8 流程优势

| 特性           | 旧流程                           | 新流程                        |
| -------------- | -------------------------------- | ----------------------------- |
| **响应时效**   | 发送后立即退出，无法处理即时回复 | 等待40s，可处理用户即时回复   |
| **对话连续性** | 单次交互                         | 支持多轮对话（最多10轮）      |

| **用户体验**   | 机器人感强                       | 更接近真人对话节奏            |

### 5.2 同步编排器 (`services/sync/orchestrator.py`)

```python
"""同步编排器 - 协调整个同步流程"""
from typing import Optional, List
from wecom_automation.core.interfaces import (
    SyncOptions, SyncResult, SyncProgress,
    IBlacklistManager, ICheckpointManager, ISyncProgressListener
)
from wecom_automation.services.sync.customer_syncer import CustomerSyncer
from wecom_automation.services.user.unread_detector import UnreadDetector
from wecom_automation.database.repository import ConversationRepository


class SyncOrchestrator:
    """
    同步编排器 - 负责协调整个同步流程

    职责:
    - 初始化同步环境
    - 获取待同步客户列表
    - 协调各组件完成同步
    - 管理同步进度和状态
    """

    def __init__(
        self,
        wecom_service,                              # WeComService
        repository: ConversationRepository,
        customer_syncer: CustomerSyncer,
        blacklist_manager: IBlacklistManager,
        checkpoint_manager: ICheckpointManager,
        unread_detector: Optional[UnreadDetector] = None,
        progress_listeners: Optional[List[ISyncProgressListener]] = None,
    ):
        self._wecom = wecom_service
        self._repository = repository
        self._customer_syncer = customer_syncer
        self._blacklist = blacklist_manager
        self._checkpoint = checkpoint_manager
        self._unread_detector = unread_detector
        self._listeners = progress_listeners or []

        self._progress = SyncProgress()
        self._current_device = None
        self._current_kefu = None

    async def run(self, options: SyncOptions) -> SyncResult:
        """
        执行同步流程

        Args:
            options: 同步选项

        Returns:
            SyncResult: 同步结果
        """
        from datetime import datetime
        start_time = datetime.now()
        errors = []

        try:
            # 1. 初始化环境
            await self._initialize()

            # 2. 获取客户列表
            customers = await self._get_customers(options)
            self._progress.total_customers = len(customers)

            # 3. 处理断点续传
            if options.resume and self._checkpoint.exists():
                customers = self._filter_resumed_customers(customers)

            # 4. 遍历同步每个客户
            for customer in customers:
                # 检查黑名单
                if self._blacklist.is_blacklisted(customer.name, customer.channel):
                    continue

                # 通知监听器
                self._notify_customer_start(customer.name)

                try:
                    # 同步单个客户
                    result = await self._customer_syncer.sync(customer, options)
                    self._update_progress(result)

                    # 保存检查点
                    self._save_checkpoint()

                except Exception as e:
                    errors.append(f"{customer.name}: {str(e)}")
                    self._notify_error(str(e))

                # 通知完成
                self._notify_customer_complete(customer.name, result.messages_count)

            # 5. 清理检查点
            self._checkpoint.clear()

        except Exception as e:
            errors.append(f"Fatal error: {str(e)}")

        return SyncResult(
            success=len(errors) == 0,
            start_time=start_time,
            end_time=datetime.now(),
            customers_synced=self._progress.synced_customers,
            messages_added=self._progress.messages_added,
            messages_skipped=self._progress.messages_skipped,
            errors=errors,
        )

    async def _initialize(self) -> None:
        """初始化同步环境"""
        # 确保WeCom已打开
        await self._wecom.ensure_open()

        # 获取当前客服信息
        kefu_info = await self._wecom.get_current_kefu()

        # 设置设备和客服
        self._current_device = self._repository.get_or_create_device(
            self._wecom.device_serial
        )
        self._current_kefu = self._repository.get_or_create_kefu(
            kefu_info.name,
            self._current_device.id,
            kefu_info.department
        )

    async def _get_customers(self, options: SyncOptions) -> List:
        """获取待同步客户列表"""
        # 导航到私聊列表
        await self._wecom.navigate_to_private_chats()

        # 提取用户列表
        users = await self._wecom.extract_all_users()

        # 未读优先排序
        if options.prioritize_unread and self._unread_detector:
            unread_users = self._unread_detector.extract()
            users = self._sort_by_unread(users, unread_users)

        # 仅未读
        if options.unread_only and self._unread_detector:
            unread_names = {u.name for u in self._unread_detector.extract()}
            users = [u for u in users if u.name in unread_names]

        return users

    def _notify_customer_start(self, name: str) -> None:
        """通知监听器：开始同步客户"""
        self._progress.current_customer = name
        for listener in self._listeners:
            listener.on_customer_start(name)
            listener.on_progress(self._progress)

    def _notify_customer_complete(self, name: str, count: int) -> None:
        """通知监听器：完成同步客户"""
        self._progress.synced_customers += 1
        for listener in self._listeners:
            listener.on_customer_complete(name, count)
            listener.on_progress(self._progress)

    def _notify_error(self, error: str) -> None:
        """通知监听器：错误"""
        for listener in self._listeners:
            listener.on_error(error)
```

### 5.3 客户同步器 (`services/sync/customer_syncer.py`)

```python
"""客户同步器 - 负责同步单个客户的对话"""
from typing import Optional, List
from dataclasses import dataclass
from wecom_automation.core.interfaces import SyncOptions, MessageContext
from wecom_automation.services.message.processor import MessageProcessor
from wecom_automation.services.user.avatar import AvatarManager
from wecom_automation.database.repository import ConversationRepository


@dataclass
class CustomerSyncResult:
    """单客户同步结果"""
    success: bool
    messages_count: int = 0
    messages_added: int = 0
    messages_skipped: int = 0
    images_saved: int = 0
    videos_saved: int = 0
    voice_count: int = 0
    error: Optional[str] = None


class CustomerSyncer:
    """
    客户同步器 - 负责同步单个客户的完整对话

    职责:
    - 进入客户对话
    - 提取历史消息
    - 调用消息处理器处理消息
    - 处理头像
    - 发送测试消息（可选）
    """

    def __init__(
        self,
        wecom_service,
        repository: ConversationRepository,
        message_processor: MessageProcessor,
        avatar_manager: Optional[AvatarManager] = None,
        ai_service = None,
        timing = None,
    ):
        self._wecom = wecom_service
        self._repository = repository
        self._message_processor = message_processor
        self._avatar_manager = avatar_manager
        self._ai_service = ai_service
        self._timing = timing

    async def sync(
        self,
        user,
        options: SyncOptions,
        kefu_id: int,
        device_serial: str,
    ) -> CustomerSyncResult:
        """
        同步单个客户的对话

        Args:
            user: 用户信息
            options: 同步选项
            kefu_id: 客服ID
            device_serial: 设备序列号

        Returns:
            CustomerSyncResult: 同步结果
        """
        result = CustomerSyncResult(success=True)

        try:
            # 1. 获取或创建客户记录
            customer = self._repository.get_or_create_customer(
                name=user.name,
                kefu_id=kefu_id,
                channel=user.channel,
            )

            # 2. 处理头像
            if self._avatar_manager:
                await self._avatar_manager.capture_if_needed(user.name)

            # 3. 进入对话
            await self._wecom.tap_on_user(user.name)
            await self._human_delay("tap")

            # 4. 提取历史消息
            messages = await self._wecom.extract_conversation_messages()

            # 5. 创建消息上下文
            context = MessageContext(
                customer_id=customer.id,
                customer_name=user.name,
                channel=user.channel,
                kefu_name="",  # 从调用方传入
                device_serial=device_serial,
            )

            # 6. 处理每条消息
            for msg in messages:
                process_result = await self._message_processor.process(msg, context)

                if process_result.get("added"):
                    result.messages_added += 1
                else:
                    result.messages_skipped += 1

                # 统计媒体类型
                if process_result.get("type") == "image":
                    result.images_saved += 1
                elif process_result.get("type") == "video":
                    result.videos_saved += 1
                elif process_result.get("type") == "voice":
                    result.voice_count += 1

            result.messages_count = len(messages)

            # 7. 发送测试消息（可选）
            if options.send_test_messages:
                await self._send_test_message(customer, context, options)

            # 8. 返回对话列表
            await self._wecom.go_back()
            await self._human_delay("tap")

        except Exception as e:
            result.success = False
            result.error = str(e)

        return result

    async def _send_test_message(
        self,
        customer,
        context: MessageContext,
        options: SyncOptions
    ) -> None:
        """发送测试消息并等待响应"""
        # 生成消息内容
        message = "测试信息: [...]"

        # 如果有AI服务，获取AI回复
        if self._ai_service:
            history = self._get_conversation_history(customer.id)
            ai_reply = await self._ai_service.get_reply(message, context, history)
            if ai_reply:
                message = ai_reply

        # 发送消息
        await self._wecom.send_message(message)

        # 等待响应
        if options.response_wait_seconds > 0:
            await self._wait_for_response(options.response_wait_seconds)

    async def _human_delay(self, delay_type: str) -> None:
        """人类行为延迟"""
        if self._timing:
            import asyncio
            delay = getattr(self._timing, f"get_{delay_type}_delay", lambda: 1.0)()
            await asyncio.sleep(delay)
```

```python
"""消息处理器 - 统一处理各类消息"""
from typing import List, Dict, Any, Optional
from wecom_automation.core.interfaces import IMessageHandler, MessageContext
from wecom_automation.database.repository import ConversationRepository


class MessageProcessor:
    """
    消息处理器 - 使用责任链模式处理不同类型消息

    职责:
    - 注册消息处理器
    - 分发消息到合适的处理器
    - 统一消息存储逻辑
    """

    def __init__(
        self,
        repository: ConversationRepository,
        handlers: Optional[List[IMessageHandler]] = None
    ):
        self._repository = repository
        self._handlers: List[IMessageHandler] = handlers or []

    def register_handler(self, handler: IMessageHandler) -> None:
        """注册消息处理器"""
        self._handlers.append(handler)

    async def process(
        self,
        message: Any,
        context: MessageContext
    ) -> Dict[str, Any]:
        """
        处理消息

        Args:
            message: 原始消息对象
            context: 消息上下文

        Returns:
            处理结果字典
        """
        # 找到能处理该消息的处理器
        for handler in self._handlers:
            if await handler.can_handle(message):
                return await handler.process(message, context)

        # 默认处理：作为文本消息
        return await self._process_as_text(message, context)

    async def _process_as_text(
        self,
        message: Any,
        context: MessageContext
    ) -> Dict[str, Any]:
        """默认文本消息处理"""
        from wecom_automation.database.models import MessageRecord, MessageType

        record = MessageRecord(
            customer_id=context.customer_id,
            content=str(message.content or ""),
            message_type=MessageType.TEXT.value,
            is_from_kefu=message.is_self,
            timestamp_raw=message.timestamp,
        )

        added, _ = self._repository.add_message_if_not_exists(record)

        return {
            "added": added,
            "type": "text",
            "content": record.content,
        }
```

```python
        return message.msg_type == "image" or (
            hasattr(message, 'image_bounds') and message.image_bounds
        )

    async def process(
        self,
        message: Any,
        context: MessageContext
    ) -> Dict[str, Any]:
        """处理图片消息"""
        from wecom_automation.database.models import MessageRecord, ImageRecord, MessageType

        # 1. 创建消息记录
        record = MessageRecord(
            customer_id=context.customer_id,
            content="[图片]",
            message_type=MessageType.IMAGE.value,
            is_from_kefu=message.is_self,
            timestamp_raw=message.timestamp,
        )

        added, msg_record = self._repository.add_message_if_not_exists(record)

        if not added:
            return {"added": False, "type": "image"}

        # 2. 下载图片
        image_path = await self._save_image(message, context.customer_id, msg_record.id)

        # 3. 创建图片记录
        if image_path:
            image_record = ImageRecord(
                message_id=msg_record.id,
                file_path=str(image_path),
                file_name=image_path.name,
            )
            self._repository.create_image(image_record)

        return {
            "added": True,
            "type": "image",
            "path": str(image_path) if image_path else None,
        }

    async def _save_image(
        self,
        message: Any,
        customer_id: int,
        message_id: int
    ) -> Path:
        """保存图片到本地"""

        filename = f"msg_{message_id}_{timestamp}.png"
        filepath = customer_dir / filename

        # 截图保存
        await self._wecom.screenshot_element(message.image_bounds, filepath)

        # 获取语音内容

        if content is None:  # SKIP
            return {"added": False, "type": "voice", "skipped": True}

        # 创建消息记录
        record = MessageRecord(
            customer_id=context.customer_id,
            content=content,
            message_type=MessageType.VOICE.value,
            is_from_kefu=message.is_self,
            timestamp_raw=message.timestamp,
            extra_info={"duration": message.voice_duration},
        )

        added, msg_record = self._repository.add_message_if_not_exists(record)

        # 触发客户语音回调
        if added and not message.is_self and self._on_customer_voice:
            self._on_customer_voice(
                context.customer_name,
                context.channel,
                context.device_serial
            )

        return {
            "added": added,
            "type": "voice",
            "content": content,
        }

    async def _get_voice_content(

        context: MessageContext

            action, text = self._interactive_handler(message)

            if action == VoiceHandlerAction.SKIP:
                return None
            elif action == VoiceHandlerAction.INPUT:
                return text
            elif action == VoiceHandlerAction.PLACEHOLDER:
                return "[语音消息]"
            elif action == VoiceHandlerAction.CAPTION:
                # 等待用户展示字幕后重新提取
                pass

        # 默认使用占位符
        return "[语音消息]"
```

```python
"""黑名单管理器"""
import json
from pathlib import Path
from typing import Optional, List, Dict
from datetime import datetime
from wecom_automation.core.interfaces import IBlacklistManager


class BlacklistManager(IBlacklistManager):
    """
    黑名单管理器 - 统一管理用户黑名单

    职责:
    - 加载/保存黑名单文件
    - 检查用户是否在黑名单
    - 添加/移除黑名单用户
    """

    def __init__(self, blacklist_file: Path):
        self._file = blacklist_file
        self._cache: Optional[Dict] = None
        self._cache_time: Optional[datetime] = None
        self._cache_ttl = 5.0  # 缓存有效期（秒）

    def _load(self) -> Dict:
        """加载黑名单"""
        # 检查缓存
        if self._cache and self._cache_time:
            if (datetime.now() - self._cache_time).total_seconds() < self._cache_ttl:
                return self._cache

        # 从文件加载
        if self._file.exists():
            try:
                with open(self._file, "r", encoding="utf-8") as f:
                    self._cache = json.load(f)
                    self._cache_time = datetime.now()
                    return self._cache
            except Exception:
                pass

        return {"users": [], "updated_at": None}

    def _save(self, data: Dict) -> bool:
        """保存黑名单"""
        try:
            data["updated_at"] = datetime.now().isoformat()
            with open(self._file, "w", encoding="utf-8") as f:
                json.dump(data, f, ensure_ascii=False, indent=2)
            self._cache = data
            self._cache_time = datetime.now()
            return True
        except Exception:
            return False

    def is_blacklisted(self, name: str, channel: Optional[str] = None) -> bool:
        """检查用户是否在黑名单"""
        data = self._load()
        for user in data.get("users", []):
            if user.get("name") == name:
                if channel is None or user.get("channel") == channel:
                    return True
        return False

    def add(
        self,
        name: str,
        channel: Optional[str],
        reason: str,
        serial: Optional[str] = None
    ) -> bool:
        """添加到黑名单"""
        data = self._load()

        # 检查是否已存在
        for user in data.get("users", []):
            if user.get("name") == name and user.get("channel") == channel:
                return False  # 已存在

        # 添加新条目
        data["users"].append({
            "name": name,
            "channel": channel,
            "serial": serial,
            "reason": reason,
            "added_at": datetime.now().isoformat(),
        })

        return self._save(data)

    def remove(self, name: str, channel: Optional[str] = None) -> bool:
        """从黑名单移除"""
        data = self._load()
        original_count = len(data.get("users", []))

        data["users"] = [
            u for u in data.get("users", [])
            if not (u.get("name") == name and (channel is None or u.get("channel") == channel))
        ]

        if len(data["users"]) < original_count:
            return self._save(data)
        return False

    def list_all(self) -> List[Dict]:
        """列出所有黑名单用户"""
        return self._load().get("users", [])
```

```python
import json
from pathlib import Path
from typing import Optional, Dict, Any, List
from datetime import datetime
from wecom_automation.core.interfaces import ICheckpointManager


class CheckpointManager(ICheckpointManager):
    """
    断点管理器 - 管理同步断点，支持断点续传

    职责:
    - 保存同步进度检查点
    - 加载检查点恢复同步
    - 清理过期检查点
    """

    def __init__(self, checkpoint_file: Path):
        self._file = checkpoint_file

    def load(self) -> Optional[Dict[str, Any]]:
        """加载检查点"""
        if not self._file.exists():
            return None

        try:
            with open(self._file, "r", encoding="utf-8") as f:
                return json.load(f)
        except Exception:
            return None

    def save(
        self,
        synced_customers: List[str],
        stats: Dict[str, int],
        kefu_name: str,
        device_serial: str,
    ) -> bool:
        """保存检查点"""
        data = {
            "synced_customers": synced_customers,
            "stats": stats,
            "kefu_name": kefu_name,
            "device_serial": device_serial,
            "timestamp": datetime.now().isoformat(),
        }

        try:
            with open(self._file, "w", encoding="utf-8") as f:
                json.dump(data, f, ensure_ascii=False, indent=2)
            return True
        except Exception:
            return False

    def clear(self) -> bool:
        """清除检查点"""
        try:
            if self._file.exists():
                self._file.unlink()
            return True
        except Exception:
            return False

    def exists(self) -> bool:
        """检查点是否存在"""
        return self._file.exists()

    def get_synced_customers(self) -> List[str]:
        """获取已同步的客户列表"""
        checkpoint = self.load()
        if checkpoint:
            return checkpoint.get("synced_customers", [])
        return []
```

```bash
  --prioritize-unread
```

### 10.2 新旧入口对比

| 特性     | `initial_sync.py` (旧)    | `initial_sync_v2.py` (新)    |
| -------- | ------------------------- | ---------------------------- |
| 架构     | 单一 `InitialSyncService` | 模块化 `SyncOrchestrator`    |
| AI 服务  | Monkey-patch 注入         | 依赖注入                     |
| Sidecar  | Monkey-patch 注入         | 依赖注入                     |
| 进度监听 | 无                        | `ISyncProgressListener` 接口 |
| 配置     | 多个参数传递              | `SyncOptions` 数据类         |
| 代码量   | ~1500 行                  | ~350 行                      |

### 10.3 关键改进

1. **依赖注入**: AI 服务和 Sidecar 客户端通过 `set_xxx` 方法注入，无需 monkey-patch
2. **进度监听**: 通过 `ISyncProgressListener` 接口接收同步进度更新
3. **清晰的配置**: 所有选项封装在 `SyncOptions` 中
4. **模块化组件**: 每个组件可独立测试和替换

### 10.4 迁移建议

1. **测试阶段**: 使用 `initial_sync_v2.py` 进行测试，确保功能一致
2. **平行运行**: 两个脚本可以并行使用，逐步迁移
3. **最终替换**: 测试通过后，用新脚本替换旧脚本
